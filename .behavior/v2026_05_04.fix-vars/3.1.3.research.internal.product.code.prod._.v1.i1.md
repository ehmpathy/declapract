**relation to wish:** this is where arrays break:

**action:** [EXTEND] — must:

2. change type to `Record<string, string | string[]>`
3. detect arrays via `Array.isArray()` and `JSON.stringify()` them

## pattern 5: replaceAll utility

**location:** `src/utils/stringPolyfill/replaceAll.ts`

**citation [9]:**
```typescript
export const replaceAll = (
  inString: string,
  matchingString: string,
  toString: string,
) => {
  return inString.replace(
    new RegExp(escapeRegExp(matchingString), 'g'),
    () => toString,
  );
};
```

**relation to wish:** accepts string as third argument. arrays must be stringified before passed.

**action:** [REUSE] — no changes needed. caller will JSON.stringify arrays.
